# ⚡ Lamda

A lightweight web project built for experimentation, learning, and rapid
front-end development.

------------------------------------------------------------------------

## 📖 Overview

Lamda is a personal web development sandbox used for experimenting with
UI ideas, JavaScript logic, and general front-end workflows.

It is intentionally simple and flexible so features can be quickly
added, changed, or removed without structural constraints.

------------------------------------------------------------------------

## 🎯 Purpose

This project is used for:

-   Learning and practicing web development
-   Testing UI/UX ideas and layouts
-   Experimenting with JavaScript functionality
-   Building reusable front-end patterns
-   Rapid prototyping of web features

## 🚀 Running the Project

### Local file (basic)

Open index.html directly in your browser.

### 🌐 Run over WiFi (server.py)

This project can be served over your local network so other devices on
the same WiFi can access it.

Run:

``` bash
python3 server.py
```

Then open:

-   On your computer: http://localhost:8000

-   On other devices (phone / laptop on same WiFi):
    http://YOUR_LOCAL_IP:8000

To find your IP:

``` bash
hostname -I
```
